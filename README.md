
# Smart Device Management System

## Project Description

This project is a Smart Device Management System developed using Python Object-Oriented Programming concepts.

The system demonstrates:

- Classes and Objects
- Constructors (__init__)
- Inheritance
- Encapsulation
- Getter and Setter Methods
- Method Overriding
- super()
- Loops
- Conditional Statements
- Menu-Driven Interface

## Devices

- SmartDevice (Parent Class)
- SmartLight
- SecurityCamera
- TemperatureSensor

## Features

- Display Device Information
- Turn Devices ON/OFF
- Read Temperature
- Increase/Decrease Brightness
- Start Recording

## How to Run

1. Open the project in Jupyter Notebook or PyCharm.
2. Run all cells.
3. Follow the menu displayed on the screen.
